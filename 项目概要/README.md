# 项目概要导航

本目录用于记录项目的说明型 Notebook，面向课程汇报、过程复盘和新成员上手。

## 文件说明

- `项目结构.ipynb`
  - 代码结构与模块关系说明
- `文件介绍.ipynb`
  - 关键文件职责、输入输出与依赖关系
- `数据集.ipynb`
  - 数据来源、组织方式、预处理与标签映射
- `命令.ipynb`
  - 常用命令汇总（训练、推理、全链路启动）
- `全流程跑通.ipynb`
  - 从数据到可视化看板的端到端执行记录

## 推荐阅读顺序

1. `项目结构.ipynb`
2. `文件介绍.ipynb`
3. `数据集.ipynb`
4. `命令.ipynb`
5. `全流程跑通.ipynb`

## 与当前代码同步建议

建议在每次关键改动后同步更新：

- 新增/删除脚本或接口
- 训练/推理参数变化
- 前端交互变化（例如 XAI 页面默认仅显示工作台）
- 输出文件路径变化
